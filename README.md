#  Shop App ✨


Shop is open-source  app for Android & ios. It is built with Flutter Framework.

Star ⭐ the repo if you like it.

## Features

 - On Boarding
 - Login 
 - Register
 - Get produits from API
 - Search for Produit
 - Add to Favorites
 - Settings
